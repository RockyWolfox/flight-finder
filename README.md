# Flight Finder

Find airports all around the world

### Version

0.0.1

## Usage

### Installation

Install the dependencies

```sh
$ npm install
```

### Serve

To serve in the browser - Runs webpack-dev-server

```sh
$ npm start
```

### Build

Compile and build

```sh
$ npm run build
```
